Why Angular JS?

Angular JS is Dynamic. We can bind elements dynamically and use them in the page without reloading the page. We can display the elements typed in the page by binding it. We don't need to reload or submit the page for validation.



Application Guidelines

  1. Enter Name. The red instruction Enter Name!!! will appear until you enter a name. Every field has similar instruction.
  2. Enter Email. The red instruction Invalid Email Address!!! will appear until you enter an email address in the format first.last@boston.gov or first.m.last@boston.gov. It won't accept a different domain address.
  3. Select Office
  4. Select Department which is a dependent dropdown box of Office.
  5. Enter Location.
  6. Enter Telephone / Extension. It takes only numbers and upto 10 digits because extension can be less than 10 digits. Characters other than numbers will see an error as Enter Numbers Only!!!
  7. Select type of leave.
  8. If you select type of leave as Other then a text box will appear below for explaination.
  9. Enter Beginning date and time of leave.
  10. Enter Ending date and time of leave.
  11. The button will remain disabled until a leave type is selected, if leave type is other then until explain box is filled, both the dates are entered and end date is after the begin date.
  12. The red instruction Enter both the dates!!! will appear only if there is not even one entry of leave period.
  13. The red instruction Add atleast 1 entry of leave!!! will appear only if there is not even one entry of leave period.
  14. Add an entry of leave to see a grid below, showing the added dates
  15. Click remove button to remove that entry.
  16. Click edit button to edit that entry.
  17. If the leave date and beginning date are same then the duration of leave is represented in hours else in days .
  18. If the vacation leave is more than a week. Then an option appears below to request for advance pay.
  19. will remain disabled until all the fields are entered without any errors. To be precise, if there is even one red instruction in the form, the submit button will remain disbaled.
  20. Click to submit.
  21. After Submitting, find the summary of the form below.
  22. Click to go back to the original form.

Description 
$scope.mailChange = function () {
    var testmail1 = /^([a-zA-Z0-9])+\.([a-zA-Z0-9])+\@boston.gov$/;

    var testmail2 = /^([a-zA-Z0-9])+\.([a-zA-Z])\.([a-zA-Z0-9])+\@boston.gov$/;

    if (!((testmail2.test($scope.emailmodel)) || (testmail1.test($scope.emailmodel))))

        $scope.mailerror = "Invalid Email Address!!!";

    else $scope.mailerror = "";
}

This function is called whenever there is a change in the email input section in the html page. This code validates the email using regex. It forces the email to be in the format of first.last@boston.gov or first.m.last@boston.gov. 

$scope.offChange = function () { 
    if ($scope.officemodel == "")
        $scope.depts = null;
    if ($scope.officemodel == "bra")
        $scope.depts = braDept;
    if ($scope.officemodel == "edic")
        $scope.depts = edicDept;
    if ($scope.officemodel == "union")
        $scope.depts = unionDept;        
}

This function is called whenever there is a change in the dropdown selection of Office. This code allocates the departments as per the Office Selected. 

$scope.teleChange = function () { 
    var testTele = /^([0-9])+$/;
    if (!(testTele.test($scope.teleModel)))
        $scope.teleError = "Enter Only Numbers!!!";
    else $scope.teleError = "";
}

This function validates the telephone number entered in the box. It shows error whenever a character other than digits are entered. 

$scope.typeChange = function () {
    if ($scope.typeModel == "other") {
        if (!$scope.expModel) {
            $scope.expError = true;
        }
        else $scope.expError = false;
    }
    else $scope.expError = false;
}

This function checks the leave type selected and if the leave type is "other" then it checks if the explain box is filled or not. 

$scope.dateChange = function () {
    if ($scope.endModel && $scope.beginModel) {
        if ($scope.endModel < $scope.beginModel) {
            $scope.dateError = 
            'End Date should be after Begin Date';
            $scope.dateDisabled = true;
        }
        else {
            $scope.dateError = '';
            $scope.dateDisabled = false;
        }
    }
}

This function checks if the end date entered is after the begin date or not. 

$scope.editClick = function (x) { 
    $scope.beginModel = $scope.dates[x].date1;
    $scope.endModel = $scope.dates[x].date2;
    $scope.typeModel = $scope.dates[x].type;
    $scope.expModel = $scope.dates[x].exp;
    $scope.dates.splice(x, 1);
}

Makes the leave entry editable, upon clicking edit button. 

$scope.reset = function () {
    $scope.empmodel = null;
    $scope.emailmodel = null;
    $scope.mailError = "";
    $scope.typeModel = null;
    $scope.expModel = null;
    $scope.officemodel = null;
    $scope.deptmodel = null;
    $scope.locModel = null;
    $scope.teleModel = null;
    $scope.beginModel = null;
    $scope.endModel = null;
    $scope.payModel = null;
    $scope.dates = [];
    $scope.submitShow = false;
}

This function resets the form.
